Simona is a Senior Cloud Developer Advocate with Microsoft Azure. She is a web developer with a passion for teaching. She spends most of her time tinkering with JavaScript in the cloud and sharing her experience with other developers at community events like meetups and conferences or online.

Talk: Build Scalable APIs using GraphQL and Serverless

Current trends in software and backend architecture is evolving towards a more loosely coupled more granular design. The latest development on that front is the advent of serverless which allows you to run applications in very cost-effective ephemeral services. This is why it is important to have a proper gateway for your API that is able to route all your requests to the designated endpoint. GraphQL stands out in that respect as being a mature open sourced standard started at Facebook.
